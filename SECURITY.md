# Security Policy

## Reporting a Vulnerability

While this is a fork of the upstream/official [golang/govulncheck-action](https://github.com/golang/govulncheck-action) repository, please don't hesitate to let me know via nick@nickfedor.com if you run across a vulnerability that warrants more than a pull request.

If you find a vulnerability within govulncheck itself, then please reference Go's [official documentation](https://go.dev/doc/security/vuln/#feedback).

Thank you!
